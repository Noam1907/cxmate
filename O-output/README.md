# O-output: Created Work

This is where everything the agents create goes — product briefs, opportunity assessments, design docs, and any other deliverable output.

---

## What Goes Here

- Product briefs (from `/prd` skill)
- Opportunity assessments (from `/mrd` skill)
- Design documents
- Any agent-created deliverable

**Note:** Daily CX intel digests go to `M-memory/intel/` — that's memory, not output.

---

## Naming Convention

Use descriptive filenames:
- `[feature-name].md` — product briefs
- `opportunity-[topic].md` — opportunity assessments
- `[topic]-[type].md` — everything else

Version files when iterating:
- `[name]-v1.md`, `[name]-v2.md`, `[name]-final.md`
