# Prompt: Close The Loop

You just created your first strategic decision brief. Now let's make the system smarter by learning from what happened. This is The Loop. It's what makes week 10 better than week 1.

Copy and paste this into your chat:

---

```
I just created my first strategic decision brief. Now I want to close The Loop so the system learns and improves.

Please do the following:

1. Read what we just created:
   - Check the latest folder in O-output/ (the one we just made)
   - Read the strategist analysis, devil's advocate review, chief of staff brief, gatekeeper review, and final decision brief

2. Read current memory and core files:
   - M-memory/learning-log.md
   - M-memory/feedback.md
   - M-memory/decisions.md
   - C-core/project-brief.md
   - C-core/voice-dna.md
   - C-core/icp-profile.md

3. Log what we learned to M-memory/learning-log.md:
   - What worked well in this decision brief? (analysis depth, risk identification, clarity of recommendation)
   - What felt off or needed revision?
   - Any patterns in how I like decisions presented?
   - What did the Gatekeeper flag?
   - What did the Devil's Advocate catch that the Strategist missed?

4. Update M-memory/decisions.md:
   - Log this decision (topic, options considered, recommendation, confidence level)
   - Note: even if I haven't decided yet, log the analysis for future reference

5. Check if anything is ready to promote to C-core:

   **From learning-log to voice-dna:**
   - Any communication pattern that worked well? Add it as a rule.
   - Any presentation style the decision-maker prefers? Capture it.

   **From decisions to project-brief:**
   - Did this analysis reveal something about business priorities or direction?

6. Sort B-brain/04-INBOX/:
   - Move any unsorted items to the right subfolder
   - Delete anything no longer relevant
   - Summarize what you sorted

7. Show me a summary:
   - What we learned from this decision brief
   - What got promoted to C-core (if anything)
   - What stayed in Memory (not ready yet)
   - What was sorted from INBOX
   - One specific thing to try differently next time

This is The Loop. Every time you close it, the system gets smarter.
```

---

## Why The Loop Matters

Most AI systems are static. You set them up and they stay the same forever.

The ABC-TOM Loop is different. Every project teaches you something. The Loop captures those lessons and bakes them into the system permanently.

**Week 1:** Memory is empty. Output is rough.
**Week 4:** Memory has patterns. Analysis improves.
**Week 8:** Strong patterns get promoted to Core. Decision briefs sound like your team.
**Week 12:** The system runs itself. You just steer.

The Loop is the difference between "using AI" and "building an AI team."

---

## When to Use This

- Right after creating any decision brief (like now!)
- At the end of each week (pair with bonus prompt 12 - weekly review)
- After a decision plays out (was the analysis right? What did we miss?)
- When your briefs start feeling generic
